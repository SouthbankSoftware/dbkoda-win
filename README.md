## Requirement
* Node 8.9.1+
* Yarn 1.3.2+
* **_optional_** windows-build-tools (some windows may need this): `yarn global add windows-build-tools` or `npm install -g windows-build-tools`

## Build from source
1. `yarn install`
2. `yarn gulp`
